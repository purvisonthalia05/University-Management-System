# University Management System

A console-based University Management System developed in Java using Object-Oriented Programming principles. The application manages student and professor records, calculates GPA, performs recursive searches, generates academic reports, and provides a menu-driven interface for university administration.

## Features

- Add and manage student records
- Add and manage professor records
- GPA calculation based on student marks
- Recursive student search by ID
- Sort students by GPA
- Sort professors by experience
- Academic report generation
- Input validation using custom exceptions
- Menu-driven user interface
